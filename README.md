## 🎮 What Is This Project?

A **retro-themed sorting visualizer** that transforms classic sorting algorithms into an engaging arcade-style experience.  
It visually demonstrates how arrays are sorted step-by-step using neon colors, animations, and interactive controls.

---

## 🧠 The Problem It Solves

Understanding sorting algorithms through theory alone can be abstract and difficult.

This project helps by:
- Making **time complexity visible**
- Showing **real-time comparisons and swaps**
- Demonstrating concepts like:
  - O(n²) inefficiency
  - Divide & Conquer strategies
  - Recursion vs Iteration

---

## 🔍 How It Works

1. 🎲 Generate a random array  
2. 🎯 Select a sorting algorithm  
3. ▶️ Start visualization  
4. 🌈 Watch color-coded operations:
   - Comparisons  
   - Swaps  
   - Sorted elements  
5. 🎉 Completion animation when sorting finishes

## 🎯 Algorithm Complexity

| Algorithm      | Best       | Average    | Worst      | Space    |
|---------------|-----------|-----------|-----------|----------|
| Bubble Sort    | O(n)       | O(n²)      | O(n²)      | O(1)     |
| Selection Sort | O(n²)      | O(n²)      | O(n²)      | O(1)     |
| Insertion Sort | O(n)       | O(n²)      | O(n²)      | O(1)     |
| Merge Sort     | O(n log n) | O(n log n) | O(n log n) | O(n)     |
| Quick Sort     | O(n log n) | O(n log n) | O(n²)      | O(log n) |
